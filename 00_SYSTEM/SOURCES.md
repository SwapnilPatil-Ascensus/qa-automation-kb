## Import Process

1. **Place raw content** in `10_IMPORTS_RAW/` in appropriate subfolder
2. **Review and extract** key information
3. **Convert to Markdown** following `02_STANDARDS/DOC_STANDARDS.md`
4. **Organize** into appropriate numbered folders (00-11)
5. **Update this file** with import details
6. **Archive** original in `10_IMPORTS_RAW/` (do not delete)
