## Learning Path

### Week 1: Foundation
1. Complete onboarding plan
2. Set up local environment
3. Understand architecture
4. Learn coding standards
5. Run first tests

2. Understand Page Objects
3. Learn API testing

4. Learn CI/CD

2. Perform RCA
3. Advanced topics
4. Contribute to improvements
5. Mentor others (if ready)
