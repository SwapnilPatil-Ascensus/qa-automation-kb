# Jira Story Template

## 1. Copy-paste template (Story description)

```markdown
## Context
[Why now? Link to Epic / initiative.]

## User outcome
As a [role], I want [capability], so that [value].

## Scope
**In:** …
**Out:** …

## Dependencies
- [ ] None
- [ ] [System/team] — owner: [name] — needed by: [date]

## Links
- Design: …
- Aha!/Confluence: …
```

## 2. Summary line format

```text
[Component] Imperative outcome (max ~80 chars)
Example: [IDP] Add forgot-password perf scenario to Jenkins job
```

## 3. WHO / WHEN

| Action | WHO | WHEN |
|--------|-----|------|
| Apply template | Creator at intake | At creation |
| Validate before refinement | PO | Before refinement invite |
| Reject incomplete | Tech Lead | At refinement if DoR fails |

## 4. Sub-task template (optional)

```text
Summary: [Verb] [object]
Description: Done when: [observable result]
Estimate: [h] or inherit from Story points split
```
