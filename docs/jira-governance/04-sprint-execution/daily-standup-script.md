# Daily Standup Script

## 1. Objective

**15 minutes max.** Surface blockers; align on sprint goal — not problem-solving.

## 2. WHEN

| Setting | Time |
|---------|------|
| Default | Same local time daily |
| Shared time zones | Window agreed by regions |

## 3. WHO facilitates

| Role | Responsibility |
|------|----------------|
| **Scrum Master** (or rotating facilitator) | Timebox, parking lot |
| **Everyone** | Answer three prompts |

## 4. Script (read aloud)

1. **Focus:** Today we align on progress toward **[Sprint Goal]**.  
2. **Round:** Each person:  
   - **Yesterday:** What did I complete toward the goal?  
   - **Today:** What will I complete?  
   - **Blockers:** Any impediment? (Jira ticket # if exists)  
3. **Parking lot:** Topics needing >2 min → note owner + time.  
4. **Close:** SM lists **actions** (owner + due).

## 5. Rules (enforce)

| Rule | Violation handling |
|------|-------------------|
| No deep design | SM interrupts → parking lot |
| No status to manager only | Speak to team |
| Blocker without Jira | Create/update before EOD |

## 6. Async option (distributed)

| Field | Post in Teams thread by [time] |
|-------|--------------------------------|
| Done / Today / Blockers | Template below |

**Async template**

```text
[Sprint Goal link]
Yesterday: …
Today: …
Blockers: … (Jira: …)
```
