# ags-automation - lighthouse

## Description
Run lighthouse reporting for Universal Enrollment

## Coverage
12/15 - Only first 2 pages of the Universal Enrollment are covered
3/6 - Updated to include 2 full flows

## Script Creation
1. General lighthouse script creation can be done through the chrome record found in the devtools
2. Lighthouse flow script examples: https://dev.to/charca/getting-started-with-lighthouse-user-flows-24dm

## Installation
Note: Lighthouse requires Node 18 LTS (18.x) or later.

1. Install Node.js from https://nodejs.org/en/ 
2. Use checked in package.json to install lighthouse and dependencies

## Execution
1. Run: node <script>
2. Example: node UE-Enrollment.js
